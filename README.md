Pathlet - Java IOC Framework in path way
=================================================
Pathlet is an enterprise ready IOC java framework. It provide flexible and dynamic mechanism to wire java beans and resource.

Pathlet框架是一个以惯例优先原则（convention over configuration）为核心设计理念的Java IOC框架；
Pathlet convention is similer to file system path convention, 
file system: 
   Each file have unique file path.
   The wildcard could be used in bath file operation.
